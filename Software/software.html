Software and Hardware I use

Here I will prowide inforamtion what software do I use and why.

Basic

Operating System
I use Arch Linux, since it works better for my daily tasks, such as programming, gaming and other text writing tasks, and has a feuture in face of god-like documentation and AUR which is incredible since there is now literally no reason to download any shady installers from internet. Also I use linux-zen kernel since it is optimized for everyday tasks.
Terminal Emulator
I use kitty since it just works out of the box.
Shell
I use bash as a shell. Since it is deadly simple and just works.
Window Manager / Desktop Environment
dwm. It is a precompiled wm written in C, as most of the things I use, it is simple and just works. As my status bar I use dwmblocks.
Text Editor
At most of my tasks I use neovim, but when it just simplier, I will use VSCodium, with neovim plugin of course.
Web Browser
Librewolf is my daily drive browser. But I have Brave installed just if something is not working right on Gecko.

File manager
Thunar and simple cd command ;)
Mail Client

Music/audio player

Of course the best one - mpv.
